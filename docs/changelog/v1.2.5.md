## 快速开始

### 安装包（推荐）

下载 `RCloneGUI-v1.2.5-setup.exe`，运行安装向导即可。安装包已内置 rclone。

### 独立可执行文件

下载 `RCloneGUI-v1.2.5-nuitka.exe` 或 `RCloneGUI-v1.2.5-pyinstaller.exe`，双击运行。
首次启动会自动从 GitHub 下载 rclone。

### 从源码运行

```bash
git clone https://github.com/GamblerIX/RCloneGUI.git
cd RCloneGUI
pip install -r requirements.txt
python main.py
```
